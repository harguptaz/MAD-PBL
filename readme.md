Clone the repo using git clone command

after cloning, use these steps to run the application:

1. Open you VS Code on the root folder.
2. Create 2 separate CMD terminals.
3. On 1st terminal, run command, "cd/client"
4. Once you are in client folder, run command, "npm run dev". The react server should start with port number displayed.
5. Click on the link to open the frontend page.
6. Now in other terminal, run command, "npm start"
7. The server should start with the port number. 
8. No need to click on the server link, just refresh the browser and the app should work fine.
